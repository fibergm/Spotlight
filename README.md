# Spotlight
This is the public repo for the Spotlight Project. 
Spotlight aims to enable home automation users, get the flux of people leaving and entering a room. 

This repo contains all files needed to build your own Spotlight from scratch as well as the documentation on how to connect it to an MQTT broker. 
The files are either in the following standard:

- Code - .ino files - Arduino IDE
- Electronic -.kicad_pcb/.kicad_sch/.kicad_pro - KiCAD 9
- 3D-Design - .step/-f3d - Any Slicer/ Autodesk Fusion
